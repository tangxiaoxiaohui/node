一.CSS3前缀 1.chroem 和 safari : -webkik 2.firefox : -moz 3.IE : -ms 4.opera : -o 二.属性 1.圆角(border-radius) 这是一个复合属性 2.盒子阴影(box-shadow) 3.文本阴影text-shadow 4.文字边界换行 5.字体 三.颜色 1.RGBA ①.R(red:红色) G(green:绿色) B(blue:蓝色) A(alpha:透明度) 2.线性渐变色 3.径向渐变色 4.边框背景色 四.css选择器 1.属性选择器 2.伪类选择器 伪元素选择器 3.not选择器(排除不要的) 4.子选择器 五.多媒体查询 1.@media all(所有多媒体类型设备)/print(打印机)/screen(电脑屏幕，平板，智能手机等)/speech(屏幕阅读器) 实例 当屏幕可视窗口尺寸大于 480 像素的设备上修改背景颜色 2.@supports 条件判断规则,根据判断的结果来配置相应的样式 六.box-sizing 1.当设置为box-sizing:content-box时，将采用标准模式解析计算，也是默认模式,在标准模式下，一个块的总宽度= width + margin(左右) + padding(左右) + border(左右). 2.当设置为box-sizing:border-box时，将采用怪异模式解析计算,在怪异模式下，一个块的总宽度= width + margin(左右)（即width已经包含了padding和border值）.
