BOM(browser:浏览器 object:对象 modle:模型) 浏览器对象模型 一.BOM关系 BOM <-- window <-- document document对象又是DOM模型的根节点 二.BOM对象的内容 1.Window 对象 JavaScript层级中的顶层对象,表示浏览器窗口 2.Navigator 对象 包含客户端浏览器信息 3.History 对象 包含了浏览器窗口访问过的url 4.Location 对象 包含了当前url的信息 5.Screen 对象 包含了客户端显示屏的信息 三.window对象 1.open(url,name,features,replace); ①.url代表新的window的地址 ②.name表示当前window的标识,如果浏览器不存在当前表示,则新创建一个窗口(window),如果浏览器(DOM)存在当前标识,则替换掉相同表示的窗口(window). ③.features参数可以用来设置浏览器的各种样式(宽高,位置,是否显示url等等) ④.replace表示是否替换当前文档(不好用). ⑤.window.open(),会有一个返回值,返回值为open的窗口对象,可以利用这个window对象来操作窗口(移动:window.moveTo(100(单位默认不用写),100),关闭:window.close()). 2.三大弹出框 ①.alert(); 警告框,只有取消按钮. ②.confirm(); 确认框,拥有取消和确定两个按钮.返回true.false ③.prompt(); 弹出输入框,拥有取消和确定两个按钮,可以获取用户的输入住,返回用户输入的内容 3.属性 window. ①.closed 返回当前窗口是否被关闭. ②.innerheight 返回当前文档显示区域的高度. ③.innerwidth 返回当前文档显示区域的宽度. ④.length 设置或返回窗口中的框架数量,当前window内部的windiw的个数. ⑤.name 设置或返回窗口的名称。 ⑥.opener 返回创建此窗口的窗口引用. ⑥.self 返回对当前窗口的引用,等价于 Window 属性. ⑦.top 返回最顶层的先辈窗口.当前window最上面的window对象. 4.属性 navigator. ①.onLine 返回指明系统是否处于脱机模式的布尔值,是否联网。 ②.appVersion 返回浏览器的平台和版本信息。 ③.userAgent 返回由客户机发送服务器的 user-agent 头部的值。(用来判断浏览器类别). ④.plugins 返回浏览器客服端安装的所有插件 5.History ①.length 返回浏览器历史列表中的 URL 数量。 ②.back() 加载 history 列表中的前一个 URL。 ③.forward() 加载 history 列表中的下一个 URL。 ④.go() 加载 history 列表中的某个具体页面。 6.属性 Location ①.hash 设置或返回从井号 (#) 开始的 URL（锚）.返回url地址中从#开始到结束的字符串. ②.host 设置或返回主机名和当前 URL 的端口号。http默认80端口,https默认443. ③.href 设置或返回完整的 URL。 ④.search 设置或返回从问号 (?) 开始的 URL（查询部分）。返回url地址中从?开始到结束的字符串. ⑤.assign() 加载新的文档。 window.location.assign("http://www.w3school.com.cn"); ⑥.reload() 重新加载当前文档。 ⑦.replace() 用新的文档替换当前文档。 7.属性 Screen ①.availHeight 返回显示屏幕的高度 (除 Windows 任务栏之外)。 ②.availWidth 返回显示屏幕的宽度 (除 Windows 任务栏之外)。 ③.width 返回显示屏幕的高度。 ④.height 返回显示屏幕的高度。
